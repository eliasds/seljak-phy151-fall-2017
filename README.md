To build locally,

    pip install python-markdown-math
    pip install mkdocs

    git clone git@github.com:bccp/seljak-phy151-fall-2017.git

    cd seljak-phy151-fall-2017

    git pull

    mkdocs build
    mkdocs gh-deploy
